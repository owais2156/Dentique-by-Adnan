## 📌 About the Project

This is a static website developed for **Dentique By Adnan**, a dental clinic. The goal of the website is to establish an online presence for the clinic, showcase services, and allow patients to easily get in touch or book appointments.

---

## 📄 Sections

| Section | Description |
|---|---|
| 🏠 **Home / Hero** | Welcome banner with clinic name, tagline, and call-to-action |
| 👨‍⚕️ **About** | Introduction to the clinic and the dentist |
| 🦷 **Services** | List of dental services offered |
| 📅 **Appointments** | Appointment booking form or contact details |
| 🖼️ **Gallery** | Photos of the clinic and procedures |

## 🛠️ Tech Stack

- **HTML5** — Page structure and content
- **CSS3** — Styling, layout, and responsiveness
- **JavaScript** — Interactivity and dynamic behavior

No frameworks or libraries required — pure vanilla web technologies.

## 🚀 Getting Started

### Prerequisites

All you need is a modern web browser. No installation or build tools required.

### Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/dentique-by-adnan.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd dentique-by-adnan
   ```

3. **Open `index.html` in your browser:**
   ```bash
   # On Windows
   start index.html

   # On macOS
   open index.html

   # Or simply double-click index.html in your file explorer
   ```
