# Dentique by Adnan — Website

Source code for the Dentique by Adnan dental clinic website (Srinagar, J&K).

Live site: https://dentiquebyadnan.netlify.app/

## Structure

- `index.html` — homepage
- `about.html` — about / team page
- `services.html` — services & pricing
- `gallery.html` — patient results gallery
- `courses.html` — dental courses / training
- `appointments.html` — appointment booking form (connects to Google Sheets + email + WhatsApp via Apps Script)
- `style.css` — shared styles
- `script.js` — shared scripts (mobile menu, scroll effects, FAQ accordion, animations)
- `icons/` — logo and image assets

## Deployment

Hosted on Netlify. Any push to `main` auto-deploys (if Git integration is connected) — otherwise, drag the project folder into Netlify's manual deploy area.
